# OSimp
- x86_32 architecture, Protected mode, Interruptable
- Boot with Grub
- Write with C and ASM

### built:
  - bootloader, linker, makefile etc.
  - GDT & IDT
  - Some software and hardware interrupts
  - Keyboard support
  - Input & Output
  - Some math and string libraries functions
  - Dynamic memory (heap)
  - Virtual memory (paging)
